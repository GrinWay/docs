In your `%kernel.project_dir%/.gitignore`

Copy down this string

```txt
/bundles/
```

And create the `%kernel.project_dir%/bundles/grinway/` directories by execution:

```console
cd "YOUR_PROJECT_PATH"
```

```console
mkdir "./bundles/grinway" -p
```

```console
cd "./bundles/grinway"
```
